# TikTok 场景模板：Before/After 对比

> 知识截止：2026-06
> 场景 ID: tiktok-before-after
> 分类：TikTok 专属 · 效果对比
> source: technical-design.md §5.3

---

## 1. 场景定义

适用于所有"前后对比"类内容：护肤效果（使用前 vs 使用后）、收纳整理（杂乱 vs 整齐）、清洁（脏污 vs 干净）、减肥/健身（before vs after）、产品效果（未使用 vs 已使用）。通过分屏对照或卡点切换，在视觉上制造强烈反差，利用人类大脑对"变化"的天然敏感度提高完播率和互动率。

**核心设计原则**：差异必须**显著且真实**——避免 100% / "instantly" 类绝对化描述；before 和 after 的画面构图、光照、角度应尽量一致，仅变化核心对比元素。

---

## 2. 关键变量列表

| 变量 | 说明 | 示例值 |
|---|---|---|
| `{before_state}` | Before 状态描述 | "杂乱衣物堆放在沙发上" / "暗沉肤色、毛孔明显" |
| `{after_state}` | After 状态描述 | "整齐叠放的衣物" / "透亮肤色、毛孔细腻" |
| `{contrast_element}` | 核心对比元素 | "衣物" / "肤色" / "油污" / "体重" |
| `{before_tone}` | Before 色调 | "偏冷色调，低饱和度" / "暗淡灰色调" |
| `{after_tone}` | After 色调 | "偏暖色调，正常饱和度" / "明亮暖色调" |
| `{presentation_mode}` | 呈现方式 | split-screen / match-cut |
| `{camera_setup}` | 镜头设置 | "静止固定镜头" / "轻微推近" |
| `{scene_context}` | 场景上下文 | "同一沙发" / "同一化妆镜前" / "同一厨房台面" |

## 3. 推荐时长

| 呈现方式 | 时长 | 说明 |
|---|---|---|
| **分屏（split-screen）** | 4-8 秒 | 左右/上下同时呈现 Before 和 After，用户可即时对比。适合静态对比（收纳、肤色变化） |
| **卡点切换（match-cut）** | 8-15 秒 | Before 先出现 3-5s，在节拍点卡点切换到 After（3-5s），最后 2s 定格在 After。适合动态对比（清洁过程、产品使用过程） |

---

## 4. 典型 shot_type

| 镜头类型 | 适用场景 | Seedance prompt 关键描述 |
|---|---|---|
| **split-screen** | 收纳、护肤效果、装修 | "分屏对照镜头：左侧画面...右侧画面...左右画面同时呈现" |
| **match-cut** | 清洁过程、美妆步骤 | "画面从 Before 状态卡点切换到 After 状态，同一角度同一构图" |
| **wipe-reveal** | 产品效果、去污 | "画面从中线向两侧擦除，露出 After 状态"（Seedance 对 wipe 支持较弱，优先使用前两种） |

## 5. Seedance prompt 示例

### 分屏版 — 收纳对比（appliance-usage 映射）— 98 字

```
分屏对照镜头：左侧画面，一堆杂乱衣物堆放在沙发上，整体偏冷色调，
低饱和度；右侧画面，同一沙发整洁叠放衣物，整体暖色调，
光线明亮。镜头静止，左右画面同时呈现 5 秒。
```

### 分屏版 — 护肤效果（cosmetic-texture 映射）— 94 字

```
分屏对照镜头：左侧画面，暗沉肤色特写，毛孔明显，偏冷灰色调；
右侧画面，同一半脸区域肤色透亮，毛孔细腻，偏暖粉色调。
镜头静止，左右画面同时呈现，柔光从正上方打亮。
```

### 卡点切换版 — 清洁效果（appliance-usage 映射）— 85 字

```
固定镜头：厨房灶台油污严重，暗淡灰色调，低饱和度，持续 3 秒。
画面卡点切至同一角度——灶台干净反光，明亮暖色调，持续 4 秒。
最后 1 秒定格在清洁后的反光细节。
```

## 6. 典型失败反例

### 失败 prompt

```
Before 时一切都非常糟糕，然后神器的清洁产品一瞬间
让所有东西变得完美无瑕，惊人的100%有效！
```

**失败原因**：
- "神器"、"完美无瑕"、"惊人" → 违反 R1 多个抽象词
- "100% 有效" → 违反合规禁词（绝对化用语）
- "一瞬间" → 违反物理合理性（清洁需要时间过程）
- "一切都非常糟糕" → 负向 prompt（应正向描述）

### 修正方向

- 将 before/after 状态都用正向具体描述（"灶台油污严重" 而非 "糟糕"）
- 去掉绝对化词汇，用具体效果词（"反光"、"透亮"）
- 明确色调差异而非抽象评价

## 8. 参数建议

### Seedance 2.0

| 参数 | 推荐值 | 理由 |
|---|---|---|
| aspect_ratio | 9:16 | TikTok 竖版，分屏时上下排列优于左右 |
| duration | 5-10s | 对比类信息密度中高，8 秒内完成最佳 |
| resolution | 1080P | 分屏时每侧画幅减半，需要基础分辨率保证 |

---

## 9. 分屏 vs 卡点切换 选择决策表

| 条件 | 推荐方式 | 理由 |
|---|---|---|
| 对比元素是静态的（收纳、肤色） | 分屏 | 用户可同时对比，无需记忆 |
| 对比元素是动态的（清洁过程、涂抹过程） | 卡点切换 | 动态过程分屏会分散注意力 |
| duration_sec < 8s | 分屏 | 卡点切换需要 before + after 各至少 3s |
| 需要强调差异量级 | 卡点切换 + 最后定格 | 定格帧给用户反应时间 |
| 产品需要展示使用过程 | 卡点切换 | 过程本身也是内容吸引力 |

---

## 10. 注意事项（Caveats）

1. **Before/After 差异必须显著且真实**：差异过小用户看不出；差异过大（"一夜变白"）违反平台合规。使用具体可观测的变化描述（"毛孔细腻"而非"皮肤变好"）。
2. **不要在同一 prompt 写多动作**："清洁+整理"拆成两个 scene，分别使用本模板。
3. **retention_tactic 推荐 `contrast`**：此模板天然匹配反差对比策略。
4. **若使用分屏，必须明确说明"左右画面同时呈现"**：Seedance 2.0 可能将"Before 然后 After"理解为前后两个镜头而非分屏。用"同时呈现"、"同一画面中"等明确措辞。
5. **After 状态不加夸张词**：不用"惊艳的"、"完美的"、"不可思议的"——用"透亮"、"整洁"、"反光"、"平整"等正向具体词。
