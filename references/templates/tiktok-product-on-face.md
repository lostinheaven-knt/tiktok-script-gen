# TikTok 场景模板：产品上脸/上身特写

> 知识截止：2026-06
> 场景 ID: tiktok-product-on-face
> 分类：TikTok 专属 · 美妆/护肤/配饰演示
> source: technical-design.md §5.2

---

## 1. 场景定义

适用于美妆（口红、粉底、眼影、腮红）、护肤（精华、面霜涂抹）、眼镜、帽子等"上脸/上身"类产品的实景演示。通过半脸特写或 45 度肖像镜头，展示产品在真实使用场景中的效果——质地、颜色、光泽和气场变化。

**核心设计原则**：产品出现在使用场景中（而非单独展示），写实感优先——避免"完美无瑕"式的修图感画面。

---

## 2. 关键变量列表

| 变量 | 说明 | 示例值 |
|---|---|---|
| `{product}` | 产品名称及核心外观 | "红色雾面口红" / "透明质酸精华液" |
| `{application_action}` | 使用动作（单动作） | "涂抹于下唇" / "轻拍于颧骨" / "戴上墨镜" |
| `{face_area}` | 面部/身体区域 | "下半脸" / "眼周" / "全脸侧影" |
| `{model_descriptor}` | 模特泛指描述 | "模特" / "一位女性" / "手部" |
| `{lighting_direction}` | 光线方向 | "左上方柔光" / "环形灯正前方" / "侧逆光勾边" |
| `{background}` | 背景处理 | "虚化为米色" / "纯白工作室" / "咖啡店窗外虚化" |
| `{product_position}` | 产品位置（用于 continuity_hint） | "画面中央偏下" / "画面右侧 1/3 处" |
| `{brand_logo_placement}` | logo 出现规则 | "产品管身自然露出" / "不单独展示 logo" |

---

## 3. 推荐时长

| 时长 | 适用场景 |
|---|---|
| **5-8 秒** | 单一产品单步骤演示（如涂口红） |
| **8-12 秒** | 产品上脸 + 效果展示（如精华吸收后光泽变化） |
| **12-15 秒** | 多步骤（分 scene 实现，每个 scene 独立一个动作） |

> 不建议超过 15 秒——美妆演示的注意力衰减快，超时后用户划走率陡升。

## 5. Seedance prompt 示例

### 美妆口红（cosmetic-texture 映射）— 81 字

```
半脸特写镜头：一只手将红色口红均匀涂抹于模特下唇，
唇部纹理清晰可见，膏体呈现哑光雾面质感。
柔光从左上方打入，面部轮廓柔和，背景虚化为米色。
镜头静止，焦点锁定唇部 5 秒。
```

### 精华涂抹（cosmetic-texture 映射）— 77 字

```
手部特写镜头：一滴透明精华液从滴管落在模特手背，
手指以打圈方式均匀抹开，精华吸收后皮肤呈现自然光泽。
环形柔光正前方，背景纯白。镜头静止，焦点锁定手背。
```

### 眼镜试戴（clothing-tryout 映射）— 74 字

```
45度侧面肖像：模特从画面外拿起一副细框墨镜，缓慢戴上，
镜片反射窗外自然光，面部表情自然放松。
侧逆光勾边勾勒面部轮廓，背景虚化为咖啡店窗景。
镜头静止 5 秒。
```

## 6. 典型失败反例

### 失败 prompt

```
完美的 4K 超清画面，一位绝美模特涂上口红，唇部完美无瑕，
惊艳的质感在电影感灯光下闪闪发光，大师之作
```

**失败原因**：
- 抽象词 >= 7（完美、4K、超清、绝美、完美无瑕、惊艳、电影感、大师之作）→ 违反 R1
- 无具体产品描述（什么口红？什么颜色？什么质地？）
- 无具体光线方向（"电影感灯光"不可操作）
- 使用了"模特"+"具体特征意图"的模糊写法 — 应该要么纯泛指"模特"，要么具体描述特征但不用真人识别信息

## 8. 合规约束

### 8.1 真人面部素材合规

Seedance 2.0 对写实真人面部素材有审核合规限制。prompt 中必须：
- 使用"模特"等泛指词汇，不使用具体真人姓名。
- 不描述可识别个人身份的独特外貌特征（如"左脸颊有一颗心形痣"）。
- 不模仿特定公众人物的外貌。

### 8.2 Logo 展示规则

若 `brand_constraint` 要求展示 logo：
- Logo 应出现在产品本体上（如口红管身的品牌刻印），通过产品本身自然露出。
- **不要单独拉镜头给 logo**——这会被 TikTok 算法标记为硬广。
- Logo 在整个 segment 中的画面占比不超过 15%。

## 10. 注意事项（Caveats）

1. **避免具体真人特征描述**：使用"模特"等泛指，不描述可识别个人的独特外貌特征。
2. **不写"完美无瑕"等抽象词**（R1）：用"唇部纹理清晰可见"、"皮肤呈现自然光泽"等正向具体描述代替。
3. **单动作 + 单运镜**：一个 prompt 只描述一个涂抹/试用动作，一个镜头状态（通常为静止）。
4. **Logo 在产品本体上展示，不单独拉镜头**：品牌露出自然融入使用场景，避免算法标记为硬广。
5. **柔光方向明确指定**：必须写清光线方向（如"左上方柔光"），这直接用于该 segment 的 `continuity_hint` 跨段衔接。
